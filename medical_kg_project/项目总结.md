# 医疗健康知识图谱实战项目总结

## 项目概览

本项目是一个完整的知识图谱实战项目，以医疗健康领域为例，展示了从数据采集到智能应用的全流程实现。项目涵盖了您要求的所有技术要点：

### 核心技术栈
- **信息抽取**：实体抽取、关系抽取、事件抽取
- **大模型应用**：本地部署ChatGLM/Qwen进行信息抽取
- **知识存储**：Neo4j图数据库
- **智能应用**：基于知识图谱的问答系统

## 项目亮点

### 1. 完整的信息抽取流程
- **多源数据采集**：支持百科爬虫、PDF解析、结构化数据导入
- **混合抽取策略**：结合规则、传统模型和大模型
- **数据标注平台**：集成Label Studio进行高效标注

### 3. 专业的图谱构建
- **本体设计**：8种节点类型，9种关系类型
- **批量导入**：支持大规模数据高效导入
- **质量控制**：医学常识检查、逻辑一致性验证

### 4. 智能问答系统
- **意图识别**：10种医疗问答意图
- **Cypher查询生成**：自动构建图数据库查询
- **答案增强**：使用大模型优化答案质量

## 快速开始指南

### 环境准备
```bash
# 1. 克隆项目
git clone https://github.com/your-username/medical-kg-project.git
cd medical-kg-project

# 2. 安装依赖
conda create -n medical_kg python=3.8
conda activate medical_kg
pip install -r requirements.txt

# 3. 启动服务
docker-compose up -d
```

### 运行演示
```bash
# 运行交互式演示
python demo.py

# 或直接运行特定功能
# 数据采集
python src/data_collection/medical_spider.py

# 启动问答系统API
python src/qa_system/medical_qa.py --mode api

# 启动Web界面
streamlit run src/web/app.py
```

### 3. Neo4j图谱管理
```python
# 创建节点

    'name': '糖尿病',
    'icd10': 'E11',

    'disease', '糖尿病',
```
