# 数据目录

这个目录用于存放知识图谱构建所需的数据文件。

## 文件格式

### train.json
训练数据文件，每行一个JSON对象，格式如下：

```json
{
  "spo_list": [
    {
      "h": {"name": "主轴电机"},
      "t": {"name": "轴承故障"},
      "relation": "部件故障"
    }
  ]
}
```

## 实体类型
- 主体
- 客体
- 部件单元
- 故障状态
- 性能表征
- 检测工具

## 关系类型
- 部件故障
- 性能故障
- 检测工具
- 组成

## 使用方法
将你的数据文件放在这个目录下，然后运行构建脚本：
```bash
python build_knowledge_graph.py
```
