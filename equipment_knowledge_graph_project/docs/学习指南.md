# 装备制造故障知识图谱构建学习指南

## 项目概述

本项目是一个完整的装备制造故障知识图谱构建学习项目，涵盖了从数据采集、标注、分析、建模到最终部署的全流程。通过这个项目，学生可以学习到：

- 实体抽取和关系抽取的方法
- 项目数据采集、标注、分析、建模和最终部署
- 本地大模型部署
- 大模型用于信息抽取的步骤
- 图谱构建与Neo4j问答逻辑

### 第四阶段：知识图谱构建

#### 4.1 Neo4j图数据库
**学习目标**：掌握图数据库的基本操作和Cypher查询语言

**核心文件**：`04_knowledge_graph/neo4j_construction/build_graph.py`

**关键技术点**：
- Neo4j数据库连接和操作
- 节点和关系的创建
- Cypher查询语言
- 图数据库约束和索引

**基本操作**：
```cypher
// 创建节点
CREATE (e:Equipment {name: "数控机床"})
CREATE (c:Component {name: "主轴"})
CREATE (f:Fault {name: "异常振动"})

// 创建关系
CREATE (c)-[:belong_to]->(e)
CREATE (f)-[:affect]->(c)

// 查询
MATCH (e:Equipment)-[:contain]->(c:Component)
RETURN e.name, c.name
```

## 常见问题与解决方案

### 1. 环境配置问题
**问题**：依赖包安装失败
**解决**：使用conda环境或检查Python版本兼容性

### 2. 模型训练问题
**问题**：显存不足
**解决**：使用梯度累积、混合精度训练或减小batch size

### 3. 数据库连接问题
**问题**：Neo4j连接失败
**解决**：检查数据库服务状态和连接参数

### 4. 大模型部署问题
**问题**：模型加载缓慢
**解决**：使用模型量化、GPU加速或分布式部署
