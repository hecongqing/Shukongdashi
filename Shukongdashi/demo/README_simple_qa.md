# 简易故障问答 DEMO

本示例演示了一个**极简版**的故障问答系统，灵感来源于 `question_wenda.py`。

## 主要文件

* `knowledge_base.json` – 迷你知识库，手工罗列了若干条
* `simple_qa_demo.py` – 解析用户问题并给出回答的脚本

## 运行方式

```bash
python simple_qa_demo.py
```

随后按照提示输入问题，例如：

* 外部24V短路会引起哪些现象？
* 手动移动X轴会遇到什么错误？
* 速度控制单元常出现哪些故障？
* ALM401报警的含义是什么？

脚本会从知识库中查找对应信息并返回结果。

> 提示：知识库数据、正则模板均可根据实际需要自行扩充。
